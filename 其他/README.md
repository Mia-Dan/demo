解决方案1：在句末输入两个空格后 再回车。[参考](https://blog.csdn.net/bawcwchen/article/details/80557442)  
本段与上一段通过「双空格」换行。

解决方案2：或者还可以直接空一行？

本段与上一段通过「空一行」换行。  

>quote 1 // 这里没加两空格0
>>quote 1.1 // 这里加了两空格1  
>>quote 1.1 // 这里没加两空格0
>>quote 1.1 // 这里没加两空格0
>>>quote 1.1.1 // 这里没加两空格0
